# Technology Competitiveness Replication Package

This repository contains the data and code necessary to reproduce all exhibits included in the accompanying report.

## Repository Structure

- `code/`: Stata do-files used for data preparation, analysis, and exhibit generation.
- `raw_data/`: Original source data downloaded from WIPO, OECD, World Bank, and PitchBook.
- `data/`: Intermediate datasets created during data processing.
- `output/`: Replicated exhibits and tables.
- `docs/`: Supplemental documentation.

## Software Requirements

All code was written and tested in Stata 19.

Required user-written package:

```stata
ssc install xtbreak
```

Some additional user-written packages may also be required. Because several were installed prior to developing this project, they may not be explicitly documented. If Stata returns an error indicating a missing package, please install the package and resume execution.

## Folder Structure

### `code/`

Contains all Stata do-files used for data preparation, analysis, and exhibit generation.

The main file is:

```text
master.do
```

Running this file will reproduce all exhibits.

### `raw_data/`

Contains the original source data used in the project.

All datasets are publicly available except for the PitchBook data, which requires institutional access.

Data sources include:

- World Intellectual Property Organization (WIPO)
- World Bank Population Estimates and Projections Database
- OECD Patent Statistics
- PitchBook

For ease of use, several source files were renamed after download. Aside from the World Bank population files, filenames may differ from those originally provided by the data source.

#### PitchBook Data

The PitchBook workbook contains:

- Five pivot tables corresponding to five separate searches
- One disclaimer worksheet

These components are stored together in a single Excel workbook.

### `data/`

Contains intermediate datasets generated from the raw data and used in exhibit production.

In particular, this folder includes the processed data used to generate **Exhibit 1**, including:

- Historical population estimates
- Population projections
- Patent application data from WIPO

### `output/`

Contains the final exhibits generated by the replication code.

#### Exhibit Notes

- **Exhibit 2** consists of two separate image files that should be combined to form the final exhibit shown in the report.
- **Exhibit 3** is generated as a LaTeX table.

## Replication Instructions

To reproduce all exhibits:

1. Open `master.do`.
2. Modify the directory paths to match your local machine.
3. The code includes conditional statements allowing multiple user directories. Before execution, specify the user profile you wish to use.
4. Run `master.do`.

The script will:

- Import and process all data
- Generate intermediate datasets
- Produce all exhibits
- Save outputs to the `output` folder

No additional scripts need to be executed manually.

## Graphical Settings

The preamble of `master.do` defines a color palette designed to be accessible to color-blind readers. All exhibits use this palette for consistency.

## Data Sources

The analysis relies on data obtained from the following sources:

### Patent Data

- World Intellectual Property Organization (WIPO), *IP Statistics Data Center*:
  - Resident patent applications
  - Patent applications per million inhabitants

- Organisation for Economic Co-operation and Development (OECD),
  *IP5 Patent Families by Technology, Priority Date and Inventor*.

### Population Data

- World Bank,
  *Population Estimates and Projections Database*.

### Venture Capital and Startup Data

- PitchBook Data, Inc.

## Data Availability

All datasets are publicly available except for PitchBook data, which require institutional access and are subject to licensing restrictions.

Users without PitchBook access may replicate all analyses that do not rely on proprietary venture capital data.

The original source filenames were modified after download for ease of use.

### PitchBook Data

PitchBook data are proprietary and may not be redistributed without authorization.

If redistribution is prohibited under your institutional license, remove the PitchBook files from the repository and replace them with a text file explaining how authorized users can obtain the data.
