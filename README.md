# Technology Competitiveness Replication Package

## Overview

This repository contains the data and code required to reproduce all exhibits presented in the accompanying report on technological competitiveness. To ensure successful replication, please preserve the folder structure exactly as provided.

---

## Repository Structure

```text

├── code/        # Stata do-files
├── raw_data/    # Original source data
├── data/        # Intermediate processed datasets
├── output/      # Final exhibits and tables
└── README.md
```

### Folder Descriptions

#### `code/`

Contains all Stata do-files used for data preparation, analysis, forecasting, and exhibit generation.

The main replication file is:

```text
master.do
```

Running this file reproduces all exhibits contained in the report.

#### `raw_data/`

Contains the original source data used in the analysis.

Data sources include:

* World Intellectual Property Organization (WIPO)
* OECD Patent Statistics
* World Bank Population Estimates and Projections Database
* PitchBook Data, Inc.

For ease of use, several downloaded files were renamed after acquisition. Consequently, filenames may differ from those provided by the original data sources.

##### PitchBook Data

The PitchBook workbook contains:

* Five pivot tables corresponding to separate searches
* One disclaimer worksheet

These components are stored together within a single Excel workbook.

#### `data/`

Contains intermediate datasets generated from the raw data.

In particular, this folder includes the processed patent and population datasets used to generate Exhibit 1.

#### `output/`

Contains all exhibits and tables generated by the replication code.

**Notes**

* Exhibit 2 consists of two separate image files that should be combined to form the final exhibit presented in the report.
* Exhibit 3 is exported as a LaTeX table.

---

## Software Requirements

All code was written and tested using **Stata 19**.

Required user-written package:

```stata
ssc install xtbreak
```

Additional user-written packages may be required. Because some packages were installed prior to development of this project, they may not be explicitly documented. If Stata reports a missing package, please install it and rerun the replication code.

---

## Replication Instructions

To reproduce all exhibits:

1. Open `master.do`.
2. Modify the directory paths to match your local machine.
3. The code includes conditional statements supporting multiple user directories. Before execution, specify the appropriate user profile.
4. Run `master.do`.

The script will automatically:

* Import and process all source data
* Create intermediate datasets
* Generate all exhibits and tables
* Save outputs to the `output/` folder

No additional scripts need to be executed manually.

## Graphical Settings

The preamble of `master.do` defines a color palette designed to be accessible to color-blind readers. All exhibits use this palette to ensure visual consistency and accessibility.

  * Resident patent applications per million inhabitants

* Organisation for Economic Co-operation and Development (OECD), *IP5 Patent Families by Technology, Priority Date and Inventor*

### Population Data

* World Bank, *Population Estimates and Projections Database*

### Venture Capital and Startup Data

* PitchBook Data, Inc.

---

## Data Availability

All datasets are publicly available except for PitchBook data, which require institutional access and may be subject to licensing restrictions.

Users without access to PitchBook may replicate all analyses that do not rely on proprietary venture capital and startup data.

If redistribution of PitchBook data is prohibited under your institutional license, the corresponding files should be removed and replaced with instructions describing how authorized users may obtain access directly from PitchBook.
